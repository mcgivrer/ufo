## Build

To build this sample:

```shell
$> npm build
```

## Run

Two options:

1. just open your favorite web browser on the index.html file, and enjoy

```shell
$> firefox ./index.html
```

2. run the project into a dev server with interactive reload:

```shell
$> npm run dev
```
